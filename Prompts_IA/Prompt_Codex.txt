Actúa como un diseñador senior de interfaces para aplicaciones de escritorio en C# Windows Forms, especializado en .NET Framework, diseño académico moderno, experiencia de usuario, formularios con Designer y uso visual de librerías como ReaLTaiizor, FontAwesome.Sharp y MaterialSkin.2.

Necesito que diseñes únicamente la interfaz gráfica de una aplicación Windows Forms para el Parcial No. 3 de Lenguaje de Programación I.

Los botones pueden quedar conectados a eventos vacíos con comentarios TODO indicando qué debe hacer cada uno.

La interfaz debe estar hecha con Windows Forms usando archivos reales del diseñador:

OBJETIVO VISUAL DE LA INTERFAZ
==============================

Diseñar una interfaz moderna, clara, académica y agradable para una aplicación llamada:

EduRegistro

Estilo visual requerido:

* Diseño moderno tipo dashboard académico.
* Colores principales: azul, blanco, violeta o verde suave.
* Buen contraste.
* Tipografía legible.
* Iconos académicos usando FontAwesome.Sharp.
* Secciones separadas mediante paneles, cards o group boxes modernos.
* Botones claros, alineados y con iconos.
* Espaciado uniforme.
* Bordes suaves.
* Nada desordenado.
* No hacer una interfaz vacía o demasiado simple.
* No hacer una interfaz sobrecargada.
* No incluir sección de estadísticas.
