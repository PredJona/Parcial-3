# Parcial3_LPI_Eq03

Entrega del Parcial No. 3 de Lenguaje de Programacion I.

## Contenido

- `P3_LPI_Eq03_RegEst`: proyecto completo en C# Windows Forms.
- `Prompts_IA`: prompt usado para generar la aplicacion con IA.
- `Evidencias`: instrucciones para capturas de pantalla.
- `CodigoPDF`: carpeta reservada para colocar los PDF de codigo fuente exportados desde Visual Studio.

## Proyecto

La aplicacion se llama `EduRegistro` y permite administrar en memoria el registro academico de estudiantes, incluyendo datos personales, carrera, modalidad, cursos inscritos y actividades extracurriculares.

## Nota importante

No se implemento la funcionalidad de estadisticas porque la guia selecciono las funcionalidades 4 y 6.
